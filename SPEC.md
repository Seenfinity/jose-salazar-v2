# Jose Salazar - Landing Page Specification

## Project Overview
- **Project Name:** Jose Salazar Portfolio Website
- **Type:** Single-page landing with scroll sections
- **Core Functionality:** Portfolio website for a classical conductor featuring biography, education, gallery, videos, schedule, and contact
- **Target Users:** Promoters, venue managers, music enthusiasts, students

## Visual & Rendering Specification

### Design Style
- **Aesthetic:** Elegant, minimalist, classical music sensibility
- **Inspiration:** diegomatheuz.com, domingohindoyan.com
- **Color Palette:**
  - Background: Deep black (#0a0a0a) with warm undertones
  - Primary text: Off-white (#f5f5f0)
  - Accent: Gold/bronze (#c9a227) for highlights
  - Secondary: Muted gray (#6a6a6a)
- **Typography:**
  - Headings: Serif font (Playfair Display or similar)
  - Body: Clean sans-serif (Inter or similar)

### Layout Structure
1. **Hero Section** - Full viewport, large photo, name, title
2. **Biography** - Text content with optional portrait
3. **Music Education** - Education philosophy and programs
4. **Gallery** - Photo grid/masonry
5. **Videos** - Embedded video player section
6. **Schedule** - Upcoming performances calendar
7. **Contact** - Contact form and info

### Sections Detail

#### Hero
- Full-height viewport
- Background: High-quality portrait from Drive
- Name: "José Salazar"
- Subtitle: "Jette Parker Ballet Conductor | Royal Opera House, London"
- Subtle fade-in animation

#### Biography
- Split layout: photo left, text right
- Full biography text from current website
- Elegant serif typography

#### Music Education
- Education philosophy
- Programs participated
- Teaching experience

#### Gallery
- Masonry or grid layout
- Lightbox for full-size view
- Photos from Drive

#### Videos
- YouTube/Vimeo embed section
- Performance videos

#### Schedule
- Upcoming performances list
- Date, venue, program

#### Contact
- Email contact
- Social links
- Simple form (optional)

## Technical Stack
- Next.js 14 (App Router)
- Tailwind CSS
- Framer Motion for animations
- Vercel deployment

## Acceptance Criteria
- [ ] Hero loads with smooth animation

- [ ] Responsive on mobile/tablet/desktop
- [ ] Gallery lightbox works
- [ ] Videos embed correctly
- [ ] Contact section functional
- [ ] Fast load times (<3s)
