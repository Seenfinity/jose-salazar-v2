# Deploy to Vercel

## Option 1: Connect GitHub (Recommended)
1. Go to https://vercel.com
2. Log in with GitHub
3. Click "Add New..." → "Project"
4. Import "jose-salazar" from GitHub
5. Click "Deploy"

## Option 2: Vercel CLI (if token is configured)
```bash
vercel --prod
```

## Project Info
- **GitHub:** https://github.com/Seenfinity/jose-salazar
- **Framework:** Next.js 16
- **Build command:** npm run build
- **Output directory:** .next
